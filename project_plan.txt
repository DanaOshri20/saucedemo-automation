```
│   └── home_page.py

│
├── config/
│   ├── config.py
│   └── .env
│
├── reports/
├── requirements.txt
└── README.md
```

### ספריות להתקנה:

```
pip install selenium pytest webdriver-manager python-dotenv allure-pytest pytest-xdist
```
